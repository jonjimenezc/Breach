# Breach

Breach is a mod inspired by Arushan's Mod Loader, based on the .asi file format. It allows you to load cars from ModLoader simply by placing Breach.asi in your game's scripts folder. Breach includes an app with features such as installing Breach.asi in your scripts folder, importing mods, naming them, saving them in a history for easy identification, and the ability to activate, deactivate, and delete them. It has a clean and organized interface, making it easy to use.

Enjoy!
